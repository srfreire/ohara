AI-powered document management and research platform for PDFs.

## Tech Stack

- **Backend**: NestJS + TypeScript + Supabase
- **Frontend**: React + Vite + TailwindCSS
- **Agent**: Python FastAPI + OpenAI

## Quick Start

```bash
# Backend
cd backend && npm install && npm run start:dev

# Frontend
cd frontend && npm install && npm run dev

# Agent (optional)
cd agent && uv sync && uv run uvicorn api.main:app --reload --port 8001
```
